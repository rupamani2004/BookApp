# Enhanced Todo List (localStorage + Search)

## Features
- Add tasks to the list
- Mark tasks as completed (line-through)
- Delete tasks
- Search tasks in real time
- Data is saved in localStorage as a JSON array
  - Each task contains: id, text, completed

## How to Use
1. Type a task in the input box and click **Add**.
2. Click on a task text to mark it as completed.
3. Click **Delete** to remove a task.
4. Use the search bar to filter tasks.
5. Tasks remain saved even after refreshing the browser.

## Technologies Used
- HTML
- CSS (Internal styling)
- JavaScript
- localStorage
